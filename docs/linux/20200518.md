::: theorem 查询端口的时候报错提示
![](/linux/1.png)

意思是缺少协议。

解决方案在 Mac 上正确使用的方法是：即-f 需要加上地址族，-p 需要加上协议 TCP 或者 UDP 等

-   如果需要查询 inet，netstat -anvf inet

-   如果需要查询 TCP， netstat -anvp tcp

-   如果需要查询 UDP，netstat -anvp udp

调整如下即可

```
netstat -anvp tcp |grep 1000
```
